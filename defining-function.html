Functions

A function definition is a regular binding where the value of the binding is a function.

For example,code defines square to refer to a function that produces the square of a given number.A function is created with an expression that starts with the keyword function . Functions have a set of parameters (in this case, only x) and a body, which contains the statements that are to be executed when the function is called. The function body of a function created this way must always be wrapped in braces, even when it consists of only a single statement.
