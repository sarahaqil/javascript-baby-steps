Javascript Data Types

Defination

The data type is a basic type of data that can be used in a program. JavaScript provides different data types to hold different types of values. The different types of data types in JavaScript are shown below:

    Primitive Data Types

  • Number: It represents numeric Value
  • String:Is a sequence of Characters
  • Boolean:Represent Boolean value either True or False
  • Undefined: It represents an undefined value
  • Null: Represents no value
