Method

Properties that contain functions are generally called 'methods' of the value they belong to, as in “toUpperCase is a method of a string”
Every string has a toUpperCase property. When called, it will return a copy of the string in which all letters have been converted to uppercase. There is also toLowerCase, going the other way.
The push method adds values to the end of an array, and the pop method does the opposite, removing the last value in the array and returning it.These names are the traditional terms for operations
on a stack. A stack, in programming, is a data structure that allows you to push values into it and pop them out again in the opposite order so that the thing that was added last is removed first.
These are common in programming—you might remember the function call stack from the which is aninstance of the same idea.
